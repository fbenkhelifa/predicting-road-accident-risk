# Data layout

This project expects Kaggle Playground Series S5E10 files under `data/raw/kaggle_s5e10/`.

## Expected files

- `train.csv`
- `test.csv`
- `sample_submission.csv`

## Notes

Competition datasets should not be redistributed in-source here.
Use the local download script to fetch data with your own Kaggle credentials.

## Quick setup (PowerShell)

Run:

- `scripts/download_data.ps1`
